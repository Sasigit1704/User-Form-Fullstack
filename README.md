# User Form Fullstack Application

This is a full-stack web application built using React, ASP.NET Core, and MongoDB.

## Features
- User form with validation (Name, Email, Phone, Gender)
- File upload functionality (PDF, DOCX, JPG, PNG)
- Frontend and Backend validation
- REST API integration
- Responsive UI design
- Loading animation and user feedback
- Data stored in MongoDB

## Tech Stack
- React.js
- ASP.NET Core Web API
- MongoDB

## Project Structure
- user-form -> React frontend
- UserFormApi -> ASP.NET Core backend

## Notes
- Files are stored on server (not in database)
- Only file path is stored in MongoDB
